# 💊 MedGuard – AI-Based Fake Medicine Detection System

## 📌 Overview

MedGuard is an AI-powered system designed to detect counterfeit (fake) medicines using machine learning techniques. The project aims to improve healthcare safety by identifying fake drugs based on data analysis and predictive modeling.

---

## 🚀 Features

* 🔍 Detects fake vs genuine medicines
* 🤖 Machine Learning-based prediction
* 📊 Data-driven analysis
* 🧠 Trained model for classification
* 💻 Easy-to-use interface
